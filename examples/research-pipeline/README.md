# Research Pipeline

> Build an automated research workflow that takes a topic, searches multiple sources, synthesizes findings, and delivers a structured report — all from a single API call.

## What You'll Build

```
Topic Input → Web Search → Knowledge Search → Agent Analysis → Report Generation → Artifact Storage
```

A 5-step workflow that:
1. Takes a research topic as input
2. Searches the web for recent information
3. Queries your knowledge graph for internal context
4. Uses a research agent to synthesize both sources
5. Generates a formatted report and stores it as an artifact

## Use Cases
- Market research automation
- Competitive analysis
- Due diligence reports
- Weekly industry digests
