# Automated Reports

> Create a workflow that generates weekly business reports automatically — pulling metrics, analyzing trends, and delivering formatted reports to your team.

## What You'll Build

```
Schedule Trigger → Pull Metrics → Analyze Trends → Generate Report → Save Artifact → Notify Team
```

A reporting automation that:
1. Defines a reusable workflow from natural language
2. Executes it with dynamic date parameters
3. Uses an agent to write narrative analysis
4. Stores reports as searchable artifacts
5. Can be scheduled via cron or external triggers

## Use Cases
- Weekly KPI dashboards
- Monthly board reports
- Daily anomaly detection summaries
- Sprint retrospective data compilation
