# Content Pipeline

> Automate content creation with a multi-stage pipeline: generate drafts, review for quality, apply brand voice, and publish — all orchestrated through LVNG workflows.

## What You'll Build

```
Brief → Draft Agent → Review Agent → Brand Voice Agent → Final Content → Artifact
```

A 5-stage content pipeline that:
1. Takes a content brief (topic, audience, format)
2. Generates a first draft with a writing agent
3. Reviews for accuracy and quality with a critic agent
4. Applies brand voice and tone guidelines
5. Produces final, publish-ready content as an artifact

## Use Cases
- Blog post generation with editorial review
- Social media content calendars
- Email newsletter automation
- Product documentation drafts
- Marketing copy with brand consistency
